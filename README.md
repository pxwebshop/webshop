# webshop
### Hướng dẫn cài đặt source code
#### Clone source code
`https://github.com/pxwebshop/webshop.git`

### Cấu hình domain với tên là
`webshop.local`

`webshop\dev\db\repository\webshop.sql`

### Đăng nhập vào admin
`http://webshop.local/wp-login`

Username: `admin`
Password: `admin`

#### Cài đặt node module
`npm install`
